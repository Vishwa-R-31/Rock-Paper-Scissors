## Game Logic
- The computer's move is randomly generated with weighted probabilities:
    - Rock: 30%
    - Paper: 40%
    - Scissors: 30%
- Score is stored using localStorage so it persists between sessions
- Real-time UI updates show:
    - Round result (Win/Lose/Tie)
    - Both players' moves
    - Score statistics
